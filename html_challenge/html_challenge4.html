List Challenge

There are 3 types of lists:

definition lists
which show terms and definitions
unordered lists
which show bullets
ordered list
which show numbers or letters

Above is an example of definition list.

Unorderd lists can have a type attribute which can be applied to the UL or the LI tag. The three values allowed are:

  • type = "square"
  • type = "disc"
  • type = "circle"

Orderd list can have a type attibute too.Used along with a start attribute they can show:

  1. lower case letters
  2. upper case
  3. lower case roman numerals
  4. upper case roman numerals
  5. numbers
